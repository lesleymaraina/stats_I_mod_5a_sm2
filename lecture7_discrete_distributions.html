Discrete Distributions

  • discrete random variable: numerical outcome that can take only countable values [i.e.: 0, 1, 2, or a finite set - CR, PR, NR]

  • discrete probability distribution: describes uncertainty about an outcome by assigning probability to each possible value via a probability mass function (PMF):

\[ p(y) = P(Y = y), \quad \sum_{y} p(y) = 1 \]

Examples:

  • binary [0/1]: variant present vs absent; AE occurred vs not (Bernoulli / Binomial)
  • counts: number of adverse events per patient; ER visits per month; reads mapping to a gene [Poisson / Negative Binomial]
  • categorical response: CR/PR/NR

Bernoulli distribution

Bernoulli distribution: models a single binary outcome

\[ Y \sim \text{Bernoulli}(p) \]

Definition

  • random variable takes values {0,1}
  • probability of success is (p)
  • probability of failure is (1-p)

Mean and variance

\[ E(Y)=p, \quad Var(Y)=p(1-p) \]

Biomedical examples

  • mutation detected vs not detected
  • disease present vs absent
  • treatment responder vs non-responder

Binomial distribution

  • binomial distribution: models the number of successes across repeated Bernoulli trials
  • binomial distribution is an extension of Bernoulli
  • binomial random variable counts successes across multiple independent Bernoulli trials

\[ Y \sim \text{Binomial}(n,p) \]

Definition

  • Counts successes in (n) independent trials
  • Each trial has success probability (p)

Mean and variance

\[ E(Y)=np, \quad Var(Y)=np(1-p) \]

Biomedical examples

  • number of responders in a cohort
  • number of mutated samples
  • sequencing read success counts

Poisson distribution

Poisson distribution: models counts of events occurring in a fixed interval

\[ Y \sim \text{Poisson}(\lambda) \]

Definition

  • counts independent events over time, space, or genomic region
  • parameter \(\lambda\) represents the expected count
  • Poisson distribution forms the basis of count regression models

Mean and variance

\[ E(Y)=\lambda, \quad Var(Y)=\lambda \]

Biomedical examples

  • Mutation counts per genome
  • Hospital admission counts
  • Rare adverse event counts

How Bernoulli, Binomial, and Poisson connect

A useful hierarchy:

  • Bernoulli: one trial, one probability \(p\)
  • Binomial: sum of \(n\) Bernoulli trials → expected count \(np\)
  • Poisson: counts per interval with expected count \(\lambda\)

Approximation (rare event limit):

  • if \(n\) is large, \(p\) is small, and \(np=\lambda\), then \(\text{Binomial}(n,p) \approx \text{Poisson}(\lambda)\)

Summary

  • discrete distributions describe uncertainty for outcomes that take countable values

  • Bernoulli: single binary outcome

  • Binomial: number of successes out of \(n\)

  • Poisson: number of events per interval

  • distributions provide interpretable parameters [probabilities and rates]

  • distributions provide a way to quantify variability across repeated cohorts and repeated sampling
