Understanding Estimates Through Uncertainty

Author

Lesley Chapman Hannah, Ph.D., M.S.

Review: Estimation in Analysis

estimation is the process of evaluating how that number behaves across repeated sampling

the following statistical estimation concepts were established:

  • population quantity [parameter] : defines the scientific target

    • example: mean eGFR, disease prevalence

  • sample-based estimator: provides a way to approximate that quantity

    • example: sample mean, sample proportion

  • point estimate : summarizes observed data into a single value

Evaluation of estimators using two key properties:

  • bias [accuracy]

    • whether the estimator is centered on the true parameter across repeated samples

  • variance [precision]

    • how much the estimate varies from one sample to another

Extending Estimation

  • previously we discussed that estimation is a task that involves using sample data to learn about an unknown population quantity
  • point estimate is single realization from a repeated-sampling process
  • estimators must be judged by their behavior across samples, including bias and variability

Current lecture addresses the following idea:

point estimate provides a single value, but scientific conclusions require an assessment of how much uncertainty surrounds that value

Extending Estimation

Lecture concepts:

  • Interval estimates extend point estimates by accounting for sampling variability, providing a range of values that are consistent with the data rather than a single value alone

  • Every estimate is subject to sampling variability, meaning that a different sample from the same population would produce a different value

  • Interval estimates provide a range that reflects this variability, indicating how stable the estimate is and how close it is likely to be to the true population quantity

Review: Point Estimates

point estimate : metric that uses sample data to produce one numerical value for a population parameter

Examples include:

  • sample mean as an estimate of a population mean
  • sample proportion as an estimate of a population proportion
  • differences in mean - observed treatment difference as an estimate of a population effect

point estimates do not show how much the estimate could vary from sample to sample

Point Estimate Limitations

Background: suppose a study reports that the sample mean eGFR is 60.8 mL/min/1.73 m²

Unaddressed questions:

  • How much would this estimate change if the study were repeated?

  • Is the mean estimated precisely, or only roughly?

  • Are nearby values such as 59, 61, or 64 still plausible for the population mean?

  • point estimates alone include no information about variability

  • two studies can report the same estimate while differing in precision

  • interval estimates provide a range that reflects variability and makes those differences visible

Interval Estimates

interval estimate :

  • range of values used to estimate a population parameter

  • extends the point estimate by giving a range of plausible values for the population parameter

Point Estimate versus Interval Estimate:

  • point estimate: best single-number summary from the sample

  • Answers: what was observed in this sample?

  • interval estimate: range of values compatible with the sample and its uncertainty**

  • Answers: what does the sample suggests about the population, allowing for sampling variability?

Defining Interval Estimates

  • interval estimate : sample-based range of values used to estimate an unknown population parameter

  • range of values computed from sample data that estimates an unknown population quantity while incorporating sampling uncertainty

Properties of interval estimates:

  • constructed from the observed sample
  • targets a population quantity, not just the sample itself
  • reflects sampling variability by indicating that the estimate could differ across repeated samples

Defining Confidence Interval

  • interval estimates involve generating a range of numbers within which the parameter is expected to lie so that within a chosen degree of confidence the true unknown parameter is capture inside the interval

  • point estimates will differ from the population parameter due to the sampling error and it is difficult to approximate how close the point estimate is to the actual parameter

  • confidence interval :

    • interval estimate constructed from sample data along with a stated confidence level

    • value generated such that over repeated samples, a specified proportion of such intervals will contain the true population parameter

Example:

  • 95% confidence interval (CI) - under repeated sampling the 95% CI captures the true population parameter about 95% of the time

Confidence Intervals and Interval Estimation

confidence interval : frequentist method for converting a point estimate and its sampling variability into an interval estimate for a population parameter

confidence intervals within the sequence of estimation:

  1. identify the population quantity of interest

Example: mean, proportion, or difference between groups

  1. compute a point estimate from the sample

a single value summarizing the observed data

  1. quantify sampling variability

assess how much the estimate would vary across repeated samples [standard error]

Confidence Intervals and Interval Estimation

confidence interval : frequentist method for converting a point estimate and its sampling variability into an interval estimate for a population parameter

confidence intervals within the sequence of estimation:

  1. construct an interval estimate

represent that variability as a range of plausible values

  1. interpret the interval estimate

use the confidence interval to identify which values of the population parameter are consistent with the observed data and to assess the precision of the estimate

Confidence Interval Versus Standard Deviation

  • standard deviation : variability among observed individual values in the sample

  • confidence interval : range for the population parameter based on the sample estimate and its uncertainty

Examples:

  • standard deviation: spread of patient-level values

e.x.: eGFR values vary greatly from patient to patient, that affects the standard deviation

  • confidence interval: plausible range for the population parameter

e.x.: how precisely the mean eGFR has been estimated

Constructing a 95% Confidence Interval

library(tidyverse)

summary_df <- ckd_df |>
  summarise(
    n = n(),
    mean_egfr = mean(egfr),
    sd_egfr = sd(egfr),
    se_egfr = sd_egfr / sqrt(n),
    lower = mean_egfr - 1.96 * se_egfr,
    upper = mean_egfr + 1.96 * se_egfr
  )

Sample Size, Standard Error, and Confidence Intervals

How Sample Size Affects Precision and Uncertainty

Standard error is defined as:

\[ SE(\bar{x}) = \frac{s}{\sqrt{n}} \]

  • as n increases → SE decreases
  • as standard error decreases → estimates vary less across repeated samples

Sample Size, Standard Error, and Confidence Intervals

Confidence intervals are constructed using the standard error:

\[ Estimate±(\text{critical value})×SE \]

  • smaller SE → narrower confidence intervals
  • larger SE → wider confidence intervals
  • larger samples → more stable estimates and tighter intervals
  • smaller samples → more variability and wider intervals
  • in research, larger studies produce more precise estimates of population quantities

Putting Together Key Concepts

  1. A point estimate gives one numerical estimate of a population quantity
  2. An interval estimate adds uncertainty by giving a range of plausible values
  3. A confidence interval is a specific kind of interval estimate with a repeated-sampling confidence interpretation
  4. Larger samples reduce the standard error \(\rightarrow\) narrowing the confidence interval and improving precision

Summary

  • statistical analysis often involves estimating population quantities using sample data

across repeated samples:

  • estimates vary due to sampling variability
  • estimators can be evaluated by:
    • bias (accuracy)
    • variance (precision)
  • point estimates summarize observed data but do not reflect uncertainty.

Summary

  • standard error (SE) quantifies estimate variability

\[ SE = \frac{s}{\sqrt{n}} \]

  • confidence intervals (CI) provide a range of plausible values:

\[ \text{Estimate} \pm (\text{critical value}) \times SE \]

  • larger sample size → smaller SE → narrower intervals
  • intervals reflect precision of the estimate
